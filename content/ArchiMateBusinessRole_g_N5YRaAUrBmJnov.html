Description

Obtain the necessary funds to pay for the services/products included in a Contract

 

 

This happens as part of Determine Acquisition Plan before Create Solicitation. The exact order in which this is done depends on the type of acquisition; MJ: seemed to a repeat. I added additional information for post-Award Contract mods; keep Budget officer.
