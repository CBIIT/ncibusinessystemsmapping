Paylist Manager is an I2E Application module that allows OEFIA users to create paylists, which CAA staff then complete and get it approved using the Paylist web Application.

 

Paylist Manager is a module that is used by specific users within the NCI OEFIA (OEFIA) to define and manage fundable Score, DOC exception selection parameters for determining PFR types and enable the creation of paylists per FY and NCAB round.

 

Core Business Functions:

  • Define and Manage Fundable Score for the current FY:
    • Provides the ability to define fundable Score for specific Cost Center/Budget Mechanism/Sub-classification, which may be further broken down by NCI Division/Office/Center and Application Type.
    • Allows users to define separate fundable Score for different RPGs - Standard, New Investigator, Early Stage Investigator, and Large (as needed).
  • Allows users to maintain the DOC Exception Selection parameters for RPGs that is used to determine the Request Type options for Funding Requests in the PFR Application.
  • Create Paylists:
    • Retrieves the applicable Grant Application for a selected FY and Round (aka. NCAB) and creates paylists for each CAA based on the defined fundable Score parameters - Cost Center & Budget Mechanism, NCI Division/Office/Center, Application Type and RPG Type
    • Support the creation of various types of RPG/Traditional Research Grant-R01/RL1 paylists based on the fundable Score and defined rules –Standard, New Investigator, Early Stage Investigator RPG, Large Paylists (as needed).
  • Ability to modify paylists or selected Grant Application on a specific paylists
  • Built-in search capabilities: Once the Paylist is generated, it allows users to view paylists created for a specific FY and Round (aka. NCAB Date).

 

 Primary Users: OEFIA staff (1 to 2 OEFIA staff are responsible to manage the fundable Score and create paylists for each NCAB round in a given FY)
