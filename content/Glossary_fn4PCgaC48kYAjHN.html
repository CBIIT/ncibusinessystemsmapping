Term - DUNS Number link

Properties
Name Value
Description The DUNS Number is a unique nine-digit number assigned by Dun and Bradstreet Information Services. It is recognized as the universal standard for identifying and keeping track of nearly 100 million businesses worldwide. Grants.gov requires a DUNS Number for registration. Organizations can secure a DUNS Number at no cost via phone at 1-866-705-5711, via email at govt@dnb.com, or via their website at http://fedgov.dnb.com/webform.
