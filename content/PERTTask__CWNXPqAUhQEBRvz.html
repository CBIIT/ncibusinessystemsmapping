Complete set of business data for CTRP. This schema will ultimately be the the relational model for CTRP. To complete this task we need to run some test cases (Top Down) and also look at current schema to make sure all data elements are captured (bottom up). I do expect that this will continuously evolve; however, this task is complete when all current data elements are captured. This tasks does not include the structured eligibility and in-eligibility criteria. Further, this tasks is not meant to imply that RedShift is fully optimized. This step is focused on creating a sane relational schema of the business data.
