  DCEG Manuscript Clearance System : Application Component

 

The Intramural system is intended to improve the efficiency of NCI  Division by providing a platform to consistently manage the execution of business processes throughout an NCI Intramural Program.  The Intramural system centrally collects and correlates fiscal and narrative project and study data in a manner that supports accurate and timely reporting and efficient strategic planning.  The system allows program staff to reconcile Publications data and process NIH reporting tasks which inherently assist with compilation of critical reports without direct interaction with the scientific faculty.

 

  DCEG Navigator : Application Component

 

The DCEG Review Navigator would lead the user to answer a series of questions about the project and generate a customized review roadmap specific to the request that will not only serve as an informational guide or checklist for the Investigator, but also as a routing sheet for each of the steps in the review workflow
