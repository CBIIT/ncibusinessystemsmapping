Description The Noncompliant Grant Application Tracking System (NGATS) form is an online system used by CSR’s Referral Officers and the ICs’ Referral Liaisons to deal with the following application-related issues:
  • Application doesn't belong with assigned IC or IRG;
  • Application overlap (duplicate/funded/premature);
  • Supposed A0 application refers to prior application:
  • Application should be changed to a different FOA;
  • Application contains text in unacceptable font;
  • Application is overstuffed;
  • Application is missing required reference letters;
  • Other.
