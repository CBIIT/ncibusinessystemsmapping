 nPrinting : Application Process

Description

nPrinting is a process from the folks at QlikView that finds section of reports specific to certain DOC from the OBF EOY Close Out Dashboard, and emails excel files with this DOC-specific information to the DOC's AO. It wakes up every 3-5 minutes, scans the dashboard db, creates excel spreadsheets, then emails the AO. QlikView assigns an identification number to each piece of data (chart, table, etc...) displayed by QlikView apps. The nPrinting process calls up specific data IDs from the OBF EOY Close Out Dashboard.

 

DOC-specific information in an excel file goes to an AO for these DOC (as of Aug 2019):

 

DOC 

-----

DCB

DCP

OD*

DCCPS

DCEG

DCTD

DEA
