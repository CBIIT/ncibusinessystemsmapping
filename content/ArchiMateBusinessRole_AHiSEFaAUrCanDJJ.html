Description
  1.   Six-digit number used to spend money in the accounting system
    ○ Working with nVision, SOFie, and Beacon
    ○ Wants to run a report from nVision and compare with SOFie in the future. Hopes that this will be possible with the FCI changes
    ○ Happens mostly in Execution
    ○ Carter does a CAN export, formats an automated query into nVision and is then automatically uploaded to the particular CANs database (active CAN database or inactive CAN database)
    ○ Maintenance involves moving CANs between active and inactive
    ○ Also goes in to update CANs in SOFie
    ○ CANs need to be added to the correct view in SOFie to make sure it is accounted for correctly
    ○ OBF prefers to use NBS and nVision, because they are more up-to-date. Some DOCs use SOFie
    SOFie has a line for incorrect codes, where the inactivated codes are stored
