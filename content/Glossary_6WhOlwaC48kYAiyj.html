Description

Coding, as defined in this document, includes four distinct processes. The R29 and third of these processes are primarily associated with extramural awards, while the second and fourth apply both to intramural projects and extramural awards.

 

  • Initial Coding upon the receipt of an Application, for the purpose of Referral to the proper Division/PO and to assist in the review process. Other codes (e.g., CAA Class Codes (PCC), which include CA and PO codes) are also applied at this stage.
  • Cross-organizational Research Coding of extramural and intramural awards. This includes the CSO, SITE, and SIC codes. RAEB applies scientific codes to extramural awards, while intramural PI apply scientific codes to their own projects.
  • Division-level Coding of extramural awards, using codes defined by a particular NCI organization, but not used across the Institute.
  • Congressionally mandated, NIH-wide automated (with the exception of a few categories) Coding for RCDC (RCDC), which categorizes all NIH research projects.

 

Each of these Coding processes is carried out through distinct systems. Initial Coding occurs through the I2E Referral and PDA modules. Cross-organizational Research Coding occurs through FLARE (for extramural), through ARS (for CCR intramural), and through the DCEG Intramural System (for DCEG intramural). Division- level Coding uses the Program Coding Module or specific other systems (e.g., PMA for DCCPS). NIH-wide categorization occurs through the (mostly) automated RCDC process/system.

For more information on RCDC Coding process, see: https://RePORT.NIH.gov/RCDC/process.aspx
