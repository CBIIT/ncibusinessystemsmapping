Description

Paylist is an I2E application module that allows selection of grants to be paid, coding of grants for POR and Basic or Applied science and allows program staff to indicate which competing grant applications should be paid.

Program tool used to document paying grants in rank order up to the payline.

 

Paylist is a module that allows NCI Extramural Program staff to view the Paylist that is created via the Paylist Manager module and select the applications either for pay or skip. The Program staff will provide the requested NCI dollar amount to support the applications that are selected for pay, which in accordance to the NCI Policy will be electronically routed to the Program, Division, Funds, and Grant Management approvals. It also supports coding of selected grants within a paylist.

 

Core Business Functions:

  • Supports electronic workflow that allows the NCI Extramural program staff to complete the selection of competing grants in a paylist for pay and/or skip. The saved paylist will be electronically routed to the Program, Division, Funds, and Grant Management approvals.
  • Provides the ability to assign grants listed in a paylist to a different paylist.
  • Coding Paylists or Individual Grant Coding: Allows coding of individual grants in a paylist (OR) en masse coding of paylists for POR (patient oriented research) and Basic or Applied science
  • Built-in search capabilities: Performs default search based on the logged-in user’s role
  • Report Generation: Generates PDF or Excel report on the search results
  • Adds the Official paycode to the system.

 

Primary Users:

  • Save/Complete Paylist: Program Directors and Other Program staff
  • Program Approval: Branch Chief, Acting Branch Chief, & Division Level Approvers
  • Division Approval: Division Director, Acting Division Director, and Division Paylist Approver Delegate
  • Funds Certifier (OEFIA)
  • Grants Management Approver (OGA)

 

Size of Customer Base:

  • NCI Program Staff
    • NCI Program Directors – 300
    • Other Program Staff – 130
  • Program level Approver - Chief/Acting Chief- 65
  • Division Level Approver:
    • Division Director/Acting Division Director - 12
    • Division Paylist Approver - 12
  • Funds Certifier (ARC & OEFIA) - 13
  • Grants Management Approver (OGA) - 10
